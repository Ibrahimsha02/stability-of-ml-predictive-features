# Stability of Machine Learning Predictive Features Under Limited Data

## 📌 Project Overview
This project analyzes the stability of machine learning predictive features when training data is limited. It compares Random Forest and Logistic Regression models using Jaccard Similarity to evaluate feature selection stability.

## 🚀 Features
- Synthetic dataset generation
- Random Forest classifier
- Logistic Regression classifier
- Feature selection stability analysis
- Jaccard Similarity
- Feature selection frequency visualization

## 🛠️ Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## ▶️ How to Run

Install the required libraries:

bash
pip install -r requirements.txt


Run the project:

bash
python feature_stability.py


## 📊 Output
- Average Stability Score
- Feature Selection Frequency Graph
- Comparison of Random Forest and Logistic Regression
